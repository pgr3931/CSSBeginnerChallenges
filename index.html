CSS Beginner Challenges

Selectors

div { /* tag selector */ } #id { /* id selector */ } .class { /* class selector */ } div.class { /* tag with specific class */ } div, span { /* multiple tags */ } * { /* universal selector */ }

Combinators

div span { /* all span descendants insides divs */ } div > span { /* all direct span children inside divs */ } div + span { /* the first span after a div */ } div ~ span { /* all spans following a div */ }

Pseudo-classes

div:hover { /* only on hover */ } div:focus { /* only when focused */ } span:first-child { /* the first span inside another element */ } span:last-child { /* the last span inside another element */ } span:nth-child(n) { /* the nth span inside another element */ } div:not(.class) { /* all divs that are not part of .class */ }

Pseudo-elements

div::before { /* inserts content before an element */ } div::after { /* inserts content after an element */ }

Functions

div { height: calc(100% - 3px) /* calculates the height responsively */ } div { height: min(100%, 3px) /* uses the smaller value */ } div { height: max(100%, 3px) /* uses the larger value */ } div { color: rgb(0, 121, 194) /* a color as rgb */ }
